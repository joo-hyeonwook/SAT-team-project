Player 클래스 - 의사코드
이동
이중배열(vector을 이용해서 5x5 행렬 만들기)을 사용해서 플레이어의 시작 위치를 설정 
Ex) arr[3][3] = 플레이어의 시작 위치
문자 WASD 중 하나를 입력창에 입력(cin) 

W  player가 있는 인덱스 값에 있는 행의 인덱스 값에 -1
Ex) arr[3][3]  W 입력  arr[2][3] 으로 플레이어 위치 이동

A  player가 있는 인덱스 값에 있는 열의 인덱스 값에 -1
Ex) arr[3][3]  A 입력  arr[3][2] 으로 플레이어 위치 이동

S  player가 있는 인덱스 값에 있는 행의 인덱스 값에 +1
Ex) arr[3][3]  S 입력  arr[4][3] 으로 플레이어 위치 이동

D  player가 있는 인덱스 값에 있는 열의 인덱스 값에 +1
Ex) arr[3][3]  D 입력  arr[3][4] 으로 플레이어 위치 이동

만약 WASD 이외의 문자 입력시  “다시 입력해주세요” 출력  다시 입력받기
그리고 행과 열의 인덱스 값이 0보다 작아지면  “오이오이 그곳은 맵이 아니라고;;” 출력  다시 입력받기




공격 & 도망
“공격은 a 도망은 b입니다 둘 중 선택하세요” 출력 & 입력 받기
a(공격)
random 함수 이용  a값을 (0~10)까지로 설정  a값 부여
a = 0 ~ 4 이면 “공격 실패” 출력  enemy 체력 그대로
a = 5 ~ 8 이면 “공격 성공” 출력  enemy 체력 감소
a = 9 ~ 10 이면 “공격 대성공” 출력  enemy 체력 대감소

b(도망)-
프로그램 종료  이동할 거 다시 실행

체력
Player 체력 변수 선언
Player 체력 = 50
공격 함수와 사용 
Enemy 공격 시
Player 체력 = 50 – Enemy 공격
만약 player 체력 < 0 이면
“넌 패배자야 쓰레기 새끼야” 출력
모든 프로그램 종료
