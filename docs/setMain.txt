printMap: 이중 반복문 사용해 타일 [ ]과 같이 출력, 플레이어/출구 인덱스는 [P]/[E]와 같이 출력.

getExit: 난수 생성 라이브러리 사용해 0-4 사이의 값 두 번 받아 출구로 지정. map[x][y]가 출구면 x*10+y 반환. Ex) 출구가 map[4][3]이면 43 반환.

getDice: 난수 생성 라이브러리 사용해 1-6 사이의 값을 두 번 받아 더한 값(2-12)을 반환.

ifBattleStart: 플레이어와 적 위치가 같을 시 battle 함수 호출.

battle: 둘 중 하나의 체력이 0이 될 때까지 행동 함수 번갈아서 호출. 만약 플레이어 체력 0이면 gameOver 함수 호출, 적 체력 0이면 전투 종료.

gameOver: 게임 오버 메시지 출력 후 파일 종료
gameClear: 게임 클리어 메시지 출력 후 파일 종료
